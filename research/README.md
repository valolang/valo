# Research

This directory contains research material, experimental modules, parser edge cases, runtime stress tests, FFI demos, interoperability experiments, and real-world prototypes used during Valo development.

These files are used to:
- validate VBA compatibility
- test runtime semantics
- stress the interpreter and FFI layer
- reproduce parser/runtime bugs
- benchmark performance
- explore future language features
- prototype APIs and module systems

Examples may include:
- Win32 desktop applications
- Discord RPC integrations
- graphics and math workloads
- callback and FFI experiments
- import system validation
- object system experiments
- structure and array stress tests

Files in this directory are considered experimental and may change frequently as the language evolves.
